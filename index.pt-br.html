Soluções práticas para cloud, automação e observabilidade.

A TrimTechBR é uma iniciativa pessoal de dois amigos — Henderson Andrade e Henrique Mizael — que constroem e abrem o código de ferramentas práticas para profissionais de cloud, times de infraestrutura e praticantes de DevOps. Nosso primeiro projeto é o AER — Azure Estate Report: dashboards HTML somente-leitura e totalmente offline do seu ambiente Azure, gerados com um único comando PowerShell.

AER — Azure Estate Report

Um módulo PowerShell 7 que conecta ao Azure com as suas credenciais existentes, coleta um inventário somente-leitura via Azure Resource Graph e APIs REST do ARM, e renderiza um relatório HTML rico, multi-página e autocontido — além de exportações em Excel e PDF.
