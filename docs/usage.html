Usage & parameters

Everything Invoke-AerReport can do — the full parameter set and the most common invocation patterns.

On this page

The command Parameters Examples Automation & -PassThru Built-in help

The command

AER exposes a single entry point, Invoke-AerReport. With no parameters it reports on every subscription in your current Azure context and writes the result to .\aer-report. 

Invoke-AerReport

Parameters

ParameterTypeDefaultDescription
-OutputPathstring.\aer-reportFolder for the HTML report (created if missing). Entry point: index.html.
-SubscriptionIdstring[](all)One or more subscription IDs to include.
-ExcludeSubscriptionNamestring[](none)Subscription display names to exclude (exact, case-insensitive).
-MaxParallelCollectorsint (1–10)4Number of collectors to run concurrently.
-SampleDataswitchoffGenerate a rich fake-data report without connecting to Azure. Useful for screenshots, demos and layout validation.
-OpenReportswitchoffOpen the report in the browser when finished.
-PassThruswitchoffReturn a summary object (paths, counts, duration).

Examples

# All subscriptions in the current context
Invoke-AerReport

# Custom output folder + open when done
Invoke-AerReport -OutputPath C:\Reports\Contoso -OpenReport

# Only specific subscriptions
Invoke-AerReport -SubscriptionId '00000000-0000-0000-0000-000000000000','1111...'

# Exclude subscriptions by name, with more parallelism
Invoke-AerReport -ExcludeSubscriptionName 'Sandbox','Lab' -MaxParallelCollectors 8

# Generate a layout/demo report with built-in sample data
Invoke-AerReport -SampleData -OutputPath .\aer-sample -OpenReport

Automation & -PassThru

Use -PassThru to capture a summary object you can use in scripts and pipelines:

$r = Invoke-AerReport -PassThru
$r.Resources       # resource count
$r.IndexHtml       # path to index.html
$r.ExcelWorkbook   # path to the XLSX export
$r.PdfReport       # path to the PDF export
Because AER is read-only and fully offline in its output, it is safe to schedule on a runner or automation host that has an authenticated Azure context with Reader access.

Built-in help

Full help is available in the console:

Get-Help Invoke-AerReport -Full
Get-Help Invoke-AerReport -Examples
