Documentation

Report reference

A guided tour of every domain in the AER dashboard. Each data table supports global search, per-column multi-select filters, column show/hide (persisted), column resize, pagination and an expandable detail accordion.

Domains

Assessment Compute Databases Applications Network Observability Policy Security Findings
Assessment

Estate-wide assessment

The entry point of the report. The overview summarizes the whole estate with KPIs and distribution charts; the inventory lists every resource with rich filtering; cloud structure maps management groups and subscriptions; and Azure Advisor surfaces Microsoft's recommendations.

Compute

Virtual machines & scale sets

Compute inventory across the tenant: virtual machines with size, OS, power state and disk details, and virtual machine scale sets with capacity and configuration.

Databases

Data services

A complete view of data services: an overview plus dedicated pages for relational databases (including SQL-on-VM), NoSQL, cache, analytics and table storage.

Applications

Application platforms

Application hosting across the estate: web / app services, functions & logic apps, containers, API & integration services, and AKS clusters — each with its own page and an overview.

Network

Networking

Networking posture: an overview with IP utilization, DNS and balancers; virtual networks with subnets, user-defined routes and an interactive peering diagram; and load balancers.

Observability

Monitoring & observability

Observability coverage: a coverage overview, diagnostic settings per resource, the Azure Monitor Agent and data-collection mapping (an interactive VM→DCR→destination diagram), and an inventory of Log Analytics workspaces and Application Insights resources.

Policy

Azure Policy & governance

Governance posture: an overview of compliance and assignments by effect, policy exemptions, and remediation tasks.

Security

Microsoft Defender for Cloud

Security posture: a Defender for Cloud overview with plan coverage, and the full list of security recommendations.

Findings

Cross-cutting findings

AER consolidates actionable findings across the estate: general security/configuration gaps, cost optimization opportunities, and observability recommendations.
