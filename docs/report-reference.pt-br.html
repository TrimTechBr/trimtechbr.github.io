Documentação

Referência do relatório

Um tour guiado por cada domínio do dashboard do AER. Toda tabela de dados suporta busca global, filtros multi-seleção por coluna, exibir/ocultar colunas (persistido), redimensionamento de colunas, paginação e um accordion de detalhes expansível.

Domínios

Assessment Compute Bancos de dados Aplicações Rede Observabilidade Policy Segurança Findings
Assessment

Assessment de todo o ambiente

O ponto de entrada do relatório. A visão geral resume todo o ambiente com KPIs e gráficos de distribuição; o inventário lista cada recurso com filtragem rica; a estrutura de cloud mapeia management groups e assinaturas; e o Azure Advisor traz as recomendações da Microsoft.

Compute

Máquinas virtuais & scale sets

Inventário de compute em todo o tenant: máquinas virtuais com tamanho, SO, estado de energia e detalhes de disco, e virtual machine scale sets com capacidade e configuração.

Bancos de dados

Serviços de dados

Uma visão completa dos serviços de dados: uma visão geral mais páginas dedicadas para bancos relacionais (incluindo SQL em VM), NoSQL, cache, analytics e table storage.

Aplicações

Plataformas de aplicação

Hospedagem de aplicações em todo o ambiente: web / app services, functions & logic apps, containers, serviços de API & integração, e clusters AKS — cada um com sua página e uma visão geral.

Rede

Redes

Postura de rede: uma visão geral com utilização de IP, DNS e balanceadores; virtual networks com subnets, rotas definidas pelo usuário e um diagrama de peering interativo; e load balancers.

Observabilidade

Monitoramento & observabilidade

Cobertura de observabilidade: uma visão de cobertura, diagnostic settings por recurso, o Azure Monitor Agent e o mapeamento de coleta de dados (um diagrama interativo VM→DCR→destino), e um inventário de workspaces Log Analytics e recursos Application Insights.

Policy

Azure Policy & governança

Postura de governança: uma visão geral de conformidade e assignments por efeito, exemptions de policy, e tarefas de remediação.

Segurança

Microsoft Defender for Cloud

Postura de segurança: uma visão geral do Defender for Cloud com cobertura de planos, e a lista completa de recomendações de segurança.

Findings

Findings transversais

O AER consolida findings acionáveis de todo o ambiente: gaps gerais de segurança/configuração, oportunidades de otimização de custo e recomendações de observabilidade.
