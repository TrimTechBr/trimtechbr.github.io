Uso & parâmetros

Tudo o que o Invoke-AerReport faz — o conjunto completo de parâmetros e os padrões de execução mais comuns.

Nesta página

O comando Parâmetros Exemplos Automação & -PassThru Ajuda embutida

O comando

O AER expõe um único ponto de entrada, Invoke-AerReport. Sem parâmetros, ele relata todas as assinaturas do seu contexto Azure atual e grava o resultado em .\aer-report. 

Invoke-AerReport

Parâmetros

ParâmetroTipoPadrãoDescrição
-OutputPathstring.\aer-reportPasta para o relatório HTML (criada se não existir). Ponto de entrada: index.html.
-SubscriptionIdstring[](todas)Um ou mais IDs de assinatura a incluir.
-ExcludeSubscriptionNamestring[](nenhuma)Nomes de exibição de assinaturas a excluir (exato, case-insensitive).
-MaxParallelCollectorsint (1–10)4Número de coletores a rodar simultaneamente.
-SampleDataswitchoffGera um relatório com dados fictícios ricos sem conectar ao Azure. Útil para screenshots, demos e validação de layout.
-OpenReportswitchoffAbre o relatório no navegador ao finalizar.
-PassThruswitchoffRetorna um objeto de resumo (caminhos, contagens, duração).

Exemplos

# Todas as assinaturas do contexto atual
Invoke-AerReport

# Pasta de saída customizada + abrir ao terminar
Invoke-AerReport -OutputPath C:\Reports\Contoso -OpenReport

# Apenas assinaturas específicas
Invoke-AerReport -SubscriptionId '00000000-0000-0000-0000-000000000000','1111...'

# Excluir assinaturas por nome, com mais paralelismo
Invoke-AerReport -ExcludeSubscriptionName 'Sandbox','Lab' -MaxParallelCollectors 8

# Gerar um relatório de layout/demo com dados de exemplo embutidos
Invoke-AerReport -SampleData -OutputPath .\aer-sample -OpenReport

Automação & -PassThru

Use -PassThru para capturar um objeto de resumo utilizável em scripts e pipelines:

$r = Invoke-AerReport -PassThru
$r.Resources       # contagem de recursos
$r.IndexHtml       # caminho do index.html
$r.ExcelWorkbook   # caminho da exportação XLSX
$r.PdfReport       # caminho da exportação PDF
Por ser somente-leitura e ter saída totalmente offline, o AER é seguro para agendar em um runner ou host de automação que tenha um contexto Azure autenticado com acesso Reader.

Ajuda embutida

A ajuda completa está disponível no console:

Get-Help Invoke-AerReport -Full
Get-Help Invoke-AerReport -Examples
